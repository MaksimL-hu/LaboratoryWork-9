# Лабораторная работа №9
## Анализ шума

---

### Исходные данные

- Входной файл: `musical-instrument-flute-rhythm-sound.wav`
- Частота дискретизации: `44100` Гц
- Исходное число каналов: `1`
- Длительность: `6.989` с
- Количество отсчетов: `308207`

---

### Метод

```text
1. Строится STFT с окном Ханна.
2. По начальному участку оценивается спектр шума.
3. Выполняется спектральное вычитание:
   Y[f,t] = max(X[f,t] - k * W[f], floor * X[f,t])
4. Фаза берется от исходного сигнала.
5. Сигнал восстанавливается через inverse STFT.
```
